# n8n Workflows

A collection of my n8n automation workflows, focused on AI-powered automations, job search pipelines, and productivity tools.

## 📂 Current Workflows

### SDE-1 Job Search Workflow (Ollama)

An AI-powered workflow that automates the process of finding Software Development Engineer (SDE-1) opportunities.

#### Features

- 🔍 Searches for SDE-1 job openings
- 🤖 Uses Ollama as the local LLM
- ⚡ Built entirely with n8n
- 🔄 Automated job discovery workflow
- 📝 Structured output for easy review

## Requirements

- n8n
- Ollama
- Required n8n community nodes (if applicable)
- Internet connection for job search APIs/websites

## Importing the Workflow

1. Download the workflow JSON.
2. Open your n8n instance.
3. Select **Import from File**.
4. Choose the workflow JSON.
5. Configure any required credentials.
6. Update the Ollama endpoint if necessary.
7. Activate the workflow.
